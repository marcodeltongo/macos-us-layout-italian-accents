# macOS US keyboard layout with accented vowels used in Italian.

A custom macOS keyboard layout based on the US layout that makes it easy to type Italian accented vowels (à, è, é, ò, ù, ì).

Perfect for Italian users who prefer the US keyboard layout but need to write in Italian.

## Installation

1. Copy the `.keylayout` file to your keyboard layouts folder:
```bash
   cp *.keylayout ~/Library/Keyboard\ Layouts/
```

2. Go to **System Preferences → Keyboard → Input Sources** and add this layout from the "Others" section

## Usage

Use `Option` + vowel to type accented characters:

| Key | Output |
|-----|--------|
| `Option + a` | à |
| `Option + e` | è |
| `Option + d` | é |
| `Option + i` | ì |
| `Option + o` | ò |
| `Option + u` | ù |

Uppercase versions work the same way with `Shift`:

| Key | Output |
|-----|--------|
| `Option + Shift + a` | À |
| `Option + Shift + e` | È |
| `Option + Shift + d` | É |
| `Option + Shift + i` | Ì |
| `Option + Shift + o` | Ò |
| `Option + Shift + u` | Ù |
